# Store Listing Images

The following images need to be added before publishing to the Play Store:

## Required
- `icon.png` — App icon, 512×512 pixels, PNG format
- `featureGraphic.png` — Feature graphic, 1024×500 pixels, PNG format

## Screenshots (minimum 2 required)
Place phone screenshots in `phoneScreenshots/` directory:
- Minimum 2, maximum 8 screenshots
- Resolution: between 320px and 3840px on any side
- Aspect ratio: 16:9 or 9:16 (portrait or landscape)
- Format: PNG or JPEG

## Notes
- Screenshots must accurately represent app functionality
- No misleading content, badges, or promotional overlays
- After adding images, update the Play Store listing via Play Console
